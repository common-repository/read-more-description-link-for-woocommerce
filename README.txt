== Description ==

Read more Description link for WooCommerce es un plugin diseñado para reducir las descripciones de productos y categorías de productos. Para tener buen posicionamiento web, se suele escribir una descripción que ocupa bastante contenido en la parte superior de las categorías de producto y los clientes tienen que hacer scroll para poder ver los productos. De igual forma, en las pestaña Descripción de productos, queda muy larga y necesitamos una manera de que la plantilla quede estéticamente mejor. Para ello hemos creado este plugin, el cual funciona de la siguiente manera:

* Al activar el plugin, veremos una nueva sección dentro de Woocommerce/Productos/Mostar, en la cual podremos activar esta opción solo para las descripciones de productos, solo para las descripciones de categorías de productos, o para ambas.
* Una vez activado, al añadir o editar un producto, veremos un nuevo campo HTML llamado "Descripción adicional". La idea es que se escriba el primer párrafo de nuestro texto en el campo que ya teníamos de "Descripción larga" y el resto del texto en el nuevo campo "Descripción adicional". Podremos hacer exactamente lo mismo desde el apartado Woocommerce/Categorías.  
* En la vista de usuario, podremos ver que aparece el primer párrafo y un enlace "Leer más". Al pulsar en dicho enlace, se despliega el resto del contenido.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for '**Read More Description link for WooCommerce**'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select '**read-more-description-link-for-woocommerce.1.0.0**.zip' from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download '**read-more-description-link-for-woocommerce.1.0.0**.zip'
2. Extract the '**read-more-description-link-for-woocommerce**' directory to your computer
3. Upload the '**read-more-description-link-for-woocommerce**' directory to the '/wp-content/plugins/' directory
4. Activate the plugin in the Plugin dashboard
